## Official & Core

- [React](https://github.com/facebook/react) — The library for web and native user interfaces.
- [React DOM](https://github.com/facebook/react/tree/main/packages/react-dom) — React package for working with the DOM.
- [React Docs](https://react.dev/) — Official React documentation with interactive examples.
- [Create React App](https://github.com/facebook/create-react-app) — Set up a modern web app by running one command.
- [React DevTools](https://github.com/facebook/react/tree/main/packages/react-devtools) — Inspect React component hierarchies in the browser.

## Routing

- [React Router](https://github.com/remix-run/react-router) — Declarative routing for React web applications.
- [TanStack Router](https://github.com/TanStack/router) — Type-safe routing for React with built-in data loading.
- [Wouter](https://github.com/molefrog/wouter) — A minimalist ~1.5KB routing library for React.
- [Next.js](https://github.com/vercel/next.js) — Full-stack React framework with file-based routing.

## State Management

- [Zustand](https://github.com/pmndrs/zustand) — A small, fast, scalable state management solution.
- [Redux Toolkit](https://github.com/reduxjs/redux-toolkit) — The official, opinionated Redux toolset for React.
- [Jotai](https://github.com/pmndrs/jotai) — Primitive and flexible state management for React.
- [Recoil](https://github.com/facebookexperimental/Recoil) — Experimental state management library for React.
- [Valtio](https://github.com/pmndrs/valtio) — Proxy-based state for React and vanilla JavaScript.
- [TanStack Query](https://github.com/TanStack/query) — Powerful async state management for server data.

## UI & Components

- [Material UI](https://github.com/mui/material-ui) — React components implementing Google's Material Design.
- [Ant Design](https://github.com/ant-design/ant-design) — Enterprise-class UI design language and React components.
- [Chakra UI](https://github.com/chakra-ui/chakra-ui) — Simple, modular and accessible component library.
- [Mantine](https://github.com/mantinedev/mantine) — Fully featured React components library with hooks.
- [Radix UI](https://github.com/radix-ui/primitives) — Unstyled, accessible UI primitives for React.
- [shadcn/ui](https://github.com/shadcn-ui/ui) — Beautifully designed components you can copy and paste.
- [React Aria](https://github.com/adobe/react-spectrum) — Adobe's library of React hooks for accessibility.

## Forms & Validation

- [React Hook Form](https://github.com/react-hook-form/react-hook-form) — Performant, flexible forms with easy validation.
- [Formik](https://github.com/jaredpalmer/formik) — Build forms in React without tears.
- [Zod](https://github.com/colinhacks/zod) — TypeScript-first schema validation library.
- [React Final Form](https://github.com/final-form/react-final-form) — High-performance subscription-based form state management.

## Animation

- [Motion](https://github.com/framer/motion) — A production-ready animation library for React.
- [React Spring](https://github.com/pmndrs/react-spring) — Spring-physics based animation library for React.
- [AutoAnimate](https://github.com/formkit/auto-animate) — Add zero-config animations to your React app.
- [Lottie React](https://github.com/Gamote/lottie-react) — Render After Effects animations in React.

## Data Fetching

- [SWR](https://github.com/vercel/swr) — React Hooks for data fetching with caching and revalidation.
- [Apollo Client](https://github.com/apollographql/apollo-client) — A comprehensive state management library for GraphQL.
- [RTK Query](https://github.com/reduxjs/redux-toolkit) — Powerful data fetching and caching tool built on Redux Toolkit.
- [Axios](https://github.com/axios/axios) — Promise-based HTTP client for browser and Node.js.

## Tables & Charts
